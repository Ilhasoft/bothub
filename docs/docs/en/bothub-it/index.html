bothub.it

bothub.it is a service maintained by Ilhasoft with all of Bothub's infrastructure, openly, free of charges and with minimal downtime.

Endpoints and environments

By accessing bothub.it you have access to Bothub Webapp

api.bothub.it

In api.bothub.it you have access to the Bothub Engine HTTP service.

nlp.bothub.it

In nlp.bothub.it you have access to the Bothub NLP HTTP service.

staging.bothub.it

staging.bothub.it is the staging environment. Here Bothub's infrastructure is hosted using its develop branch source code.

In staging.bothub.it you have access to the Bothub Webapp.

In staging.bothub.it:8000 you have access to the Bothub Engine HTTP service.

In staging.bothub.it:2657 you have access to the Bothub NLP HTTP service.

docs.bothub.it

In docs.bothub.it you have access to the Bothub Docs.
